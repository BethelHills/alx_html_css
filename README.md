## ğŸš¦ Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, or Edge)
- A code editor (VS Code, Sublime Text, etc.)
- Basic knowledge of HTML and CSS

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/BethelHills/alx_html_css.git
   ```

2. **Navigate to the project directory:**
   ```bash
   cd alx_html_css
   ```

3. **Open with a local server (recommended):**
   ```bash
   # Using Python 3
   cd css_advanced
   python3 -m http.server 8000
   
   # Then open http://localhost:8000 in your browser
   ```

4. **Or simply open the HTML file:**
   ```bash
   # On macOS
   open css_advanced/index.html
   
   # On Linux
   xdg-open css_advanced/index.html
   
   # On Windows
   start css_advanced/index.html
   ```
