How to Use

1. Include the CSS file in your HTML:

<link rel="stylesheet" href="holberton_school-icon.css">

2. Use the icon class on a span or i element:

<span class="holberton_school-icon-ic_sound"></span>

3. Style the icon with CSS:

.my-icon {
  font-size: 64px;
  color: #ff6565;
  display: block;
  margin: 20px auto;
}

4. Example Implementation:

<div class="feature">
  <span class="holberton_school-icon-ic_sound feature-icon" aria-hidden="true"></span>
  <h3>Sound Quality</h3>
  <p>Experience crystal clear audio</p>
</div>
